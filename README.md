# 🚀 Como começar?

```sh
  git clone https://github.com/endriwmsi/ignitelab.git
  cd ignitelab
```

```sh
  npm i # com NPM
  yarn # com Yarn
```

```sh
  npm run dev # com NPM
  yarn dev # com Yarn
```
